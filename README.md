# Folder Organiser

A Python CLI tool to automatically organise files in a folder into categories based on file type.


# Features

- Organises files into categories:
- Images, Documents, Audio, Video, Code, Archives, No Extension, Others
- Handles **duplicate filenames** by appending `(1)`, `(2)`, etc.
- Ignores subfolders (only sorts files in the given folder)
- Provides a **summary of files moved**
- Fully **interactive CLI** with input validation
- Extensible: Add your own file type categories in `logger.py`
